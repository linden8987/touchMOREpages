About touchMORE

touchMORE is a high-level emulator (HLE) written in Rust. It is designed to run early iPhone OS games on modern systems by reimplementing iOS frameworks in a clean-room environment.

  • Supports iPhone OS 2.x and 3.0 binaries.
  • Compatible with Windows, macOS (Intel/Apple Silicon), and Android.
  • Actively working toward 64-bit iOS support.

Usage

Drop your .ipa or .app files into the touchHLE_apps folder. On Android, the path is typically:
/sdcard/Android/data/org.touchhle.android/files/touchHLE_apps
