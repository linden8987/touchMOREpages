Project Overview

touchMORE is a community-driven fork of touchHLE, focusing on high-level emulation (HLE) of early iPhone OS apps. Written in Rust, it allows you to run classic iOS games natively on modern desktop and mobile hardware.

  • Optimized for iPhone OS 2.x and 3.0.
  • Experimental work for 64-bit iOS support.

Platform Support

Binary releases are available for:

  • Windows (x64)
  • macOS (x64 / Apple Silicon)
  • Android (AArch64)

Usage

To run a game, place your decrypted .ipa or .app bundles in the touchHLE_apps folder. Launch the emulator and select your app from the picker.

Command Line

./touchHLE "Game.ipa" --allow-network-access

Android Specifics

Place your apps in the following directory:

/sdcard/Android/data/org.touchhle.android/files/touchHLE_apps
