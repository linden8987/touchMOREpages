Project Overview

touchMORE is a community fork of touchHLE, written in Rust. It functions as a high-level emulator, providing system frameworks to run early iPhone OS software on modern platforms.

  • Reimplements system frameworks natively.
  • Focuses on iPhone OS 2.x and 3.0 titles.
  • Developed for Windows, macOS, and Android.

Usage

Place .ipa or .app bundles in the touchHLE_apps folder. On Desktop systems, the emulator will present an app picker on startup.

Android Notes

Install the APK and place games in the following directory:

/sdcard/Android/data/org.touchhle.android/files/touchHLE_apps
